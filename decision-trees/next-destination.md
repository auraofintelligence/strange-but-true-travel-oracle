# Decision Tree: Next Destination

## Question

Where should Luke go next?

## Step 1: Remove Hard Stops

Stop if:

- official advice says do not travel
- visa path is impossible or unclear
- budget cannot survive it
- insurance is legally or contractually required and Luke cannot or will not meet that framework
- live safety risk is too high
- the main reason is panic, pressure, or fantasy urgency

## Step 2: Score Survivability

Rate 0 to 5:

- safety
- visa simplicity
- cost
- route feasibility
- health and recovery
- communications

Any score below 2 needs review before continuing.

## Step 3: Score Strange But True Fit

Rate 0 to 5:

- service usefulness
- writing value
- connection value
- learning value
- future project value
- public-safe story value
- Shared Table and cultural context value
- serendipity and spontaneity value
- romance/intercultural relationship context

## Step 4: Add Pattern Notes

Ask:

- Is there a repeated symbol, dream, timing pattern, or body signal?
- Does it suggest a practical check?
- Does it create a useful caution?

## Step 5: Check the Shared Table Lens

Ask:

- What does hospitality look like here?
- Who already feeds, gathers, rescues, cooks, hosts, or teaches?
- What needs consent or cultural guidance?
- What should not be public?

## Step 6: Check the Serendipity and Relationship Lens

Ask:

- What useful surprise, invitation, event, route, or person is appearing?
- What is the smallest safe spontaneous action?
- Is romance or intercultural connection part of the field?
- What consent, legal, cultural, language, age, money, visa, or power-difference issue matters?
- Does GGM help make sense of the pattern, or does it risk projecting theory onto real people?

## Step 7: Choose Status

- `go-now`
- `prepare`
- `watch`
- `pause`
- `no-go`

## Step 8: Name the Next Action

Examples:

- check official visa page
- check whether insurance proof is legally or contractually required
- message contact
- price route
- wait 7 days
- write country dossier
- do not proceed
