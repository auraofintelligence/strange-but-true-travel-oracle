# Private Boundary

This repo is private by default.

## Keep Private

- Exact live location.
- Passport, visa application, banking, insurance, and health details.
- Private contacts and invitations.
- Private logistics such as accommodation addresses and booking references.
- Travel dates before they are safe to share.
- Personal fears, dreams, intuition, and pattern notes.
- Romance, attraction, dating, intimacy, relationship, and family notes unless deliberately redacted.
- Private GGM / Global Group Marriages reflections that refer to real people.
- Agent prompts that include sensitive personal context.
- Any source file from the private field library unless it has been redacted.

## Can Become Public Later

Only after deliberate review:

- A high-level travel story.
- A public Strange But True banner page.
- A sanitised country reflection.
- A sample decision tree with private data removed.
- A public lesson about responsible AI, travel planning, or local service.

## Never Publish

- Passport numbers.
- Private wallet keys, seed phrases, balances, or full transaction histories.
- Passwords, account recovery information, or one-time codes.
- Full private itinerary while travel is active.
- Personal safety vulnerabilities.
- Unverified claims presented as fact.

## Precognition Boundary

Precognition/planning material may be useful as a private signal log. It is not evidence by itself.

Write it as:

- "Pattern note"
- "Dream note"
- "Intuition note"
- "Scenario to watch"
- "If this repeats, check..."

Do not write it as:

- "This will happen"
- "The oracle proved..."
- "Guaranteed"
- "Confirmed by signs"

## Redaction Pass

Before pushing, sharing, exporting, or copying:

1. Search for absolute Windows paths.
2. Search for personal identifiers.
3. Search for dates, addresses, booking references, and live route details.
4. Remove or generalise sensitive material.
5. Add a note saying what was redacted and why.
