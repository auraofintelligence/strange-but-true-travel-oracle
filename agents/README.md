# Agent Team Charter

The Travel Oracle works through small specialist agents.

Each agent produces markdown notes. The Synthesis Agent combines them into a dated oracle run.

## Handoff Pattern

1. Intake frames the question.
2. Safety checks whether the idea is even sensible.
3. Visa and Logistics checks entry feasibility.
4. Route and Budget checks survival.
5. Serendipity checks useful surprise, timing, invitation, drift and smallest safe spontaneous action.
6. Romance and Intercultural Relationships checks attraction, consent, legal/cultural context, kinship and GGM relevance.
7. Opportunity checks why this matters under the Strange But True banner.
8. Shared Table checks food, hospitality, cultural context, consent, and reciprocity.
9. Precognition and Planning adds private pattern notes and future scenarios.
10. Synthesis creates the recommendation.
11. Privacy reviews before anything leaves the repo.

## Output Rule

Every agent note should end with:

- `Recommendation`
- `Risks`
- `Open questions`
- `Next action`
