# Visa and Logistics Agent

## Purpose

Work out whether Luke can legally and practically enter, stay, work, meet, create, and leave.

## Checks

- visa-free, visa on arrival, e-visa, embassy visa, business visa
- passport validity
- onward ticket requirements
- vaccination or health documents
- customs rules for gear
- local transport on arrival
- accommodation proof
- entry forms
- processing time
- cost

## Output

- Entry status
- Visa path
- Required documents
- Processing time
- Cost estimate
- Arrival risks
- Source notes

## Rule

If the source is not official, mark it as `operational` or `unknown`, not `official`.
