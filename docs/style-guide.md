# Style Guide

## Voice

Plain Australian English.

Write like a calm field notebook:

- clear
- practical
- honest about uncertainty
- not overhyped
- not mystical as proof
- not corporate
- respectful of cultural context

## Banner

The public banner is Strange But True.

The private voice can be more direct:

- "Luke is travelling under the Strange But True banner."
- "This trip should produce useful work, not just motion."
- "The oracle can suggest, but Luke decides."

## Interface Feel

Modern, private, calm, and readable.

Avoid a generic tourist-site feel. This is more like a quiet mission dashboard:

- dark base
- readable cards
- clear status badges
- maps and tables where useful
- warning colours only for warning states
- no clutter

## Colour Notes

- Night ink: private dashboard base.
- Cyan: navigation and live checks.
- Green: ready/safe enough.
- Gold: insight, opportunity, Strange But True fit.
- Orange: caution.
- Red: stop/no-go.

## Copy Rules

- Do not call this a public travel-advice service.
- Do not call it a paid bundle.
- Do not promise real-time accuracy unless the source was checked live.
- Do not say a pattern note predicts the future.
- Do not treat food, ceremony, hospitality, or cultural protocol as content to harvest.
- Always separate official evidence from intuition.
